Package: edu.isu.xrese.server.base

[Groovy] Class BaseServer

    • edu.isu.xrese.server.base.BaseServer

    • Class containing the basic logic needed to setup and get a server running. This does not contain any route information. Rather it leaves this to the child classes. Note that this is an implementation of the Template Method pattern.
